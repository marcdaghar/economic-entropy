# Economic Entropy and Thermodynamic Stability

This repository contains the code for the paper "Economic Entropy and the Thermodynamic Foundations of Monetary Stability" by Marc Daghar.

## Overview

The framework implements:
1. Production entropy S_prod = S_min + η·Q(t)
2. Negentropy capture S_neg = γ·dV_stock/dt
3. Total entropy dynamics
4. Stability proof via Lyapunov methods
5. Relationship between Λ and entropy
